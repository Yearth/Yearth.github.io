解析

题目的需求是比较明确的，这里我们来逐步分析一下：

  • 首先 timeout 的功能是在 ms 的延迟之后进行 resolve
  • 那么 ajax1, ajax2, ajax3 的功能则分别是在 2, 1, 2 s 的延迟后输出 1, 2, 3

那么我们先来依次调用一下 ajax1, ajax2, ajax3 看看结果：

这个其实很好解释，因为 js 是单线程的，遇到异步任务直接扔进队列中等待执行，那么这里的执行流程如下：

  • 遇到 ajax1，扔进队列，等待时间为 2s
  • 遇到 ajax2，扔进队列，等待时间为 1s
  • 遇到 ajax3，扔进队列，等待时间为 2s
  • 1s，ajax2 执行完毕，输出 2
  • 2s，ajax1 和 ajax3 执行完毕，因为 ajax1 先入队，所以输出 1, 3

那么题目的意思就很明显了，将并行改为串行，依次输出 1, 2, 3。至于后面的 done [1, 2, 3] 就相对简单了。
